Downloading and Installation Guide

The BLAST packages should be downloaded through anonymous ftp using an ftp client, or other tools such as a web browser, wget, curl, etc. In this guide we will use an ftp download process using the traditional ftp client in a Linux environment.

Downloading steps

1. Connect using FTP

ftp ftp.ncbi.nlm.nih.gov (user:anonymous pswd:your email)

2. Go to BLAST directory

fcd blast/executables/LATEST/

3. With the command ls you can find the latest realease. Get the latest realease using these commands:

fget ncbi-blast-2.6.0+-x64-linux.tar.gz

get ncbi-blast-2.6.0+-x64-linux.tar.gz.md5

4. Quit

quit

Installation Guide

To install, simply extract the downloaded package after placing it under a desired directory. This can be accomplished by a single tar command, or a combination of gunzip and tar commands.

$ tar zxvpf ncbi-blast+2.6.0+-x64-linux.tar.gz

or

$ gunzip -d ncbi-blast-2.6.0+-x64-linux.tar.gz

$ tar xvpf ncbi-blast-2.6.0+-x64-linux.tar

Configuration

Using the BLAST+ package installed above without configuration could be cumbersome â€“ it requires the installation path to be prefixed to the program call and database specification since the system does not know where to look for the installed program and the specified database. To streamline BLAST searches, two environment variables, PATH and BLASTDB, need to be modified and specified, respectively, to point to the corresponding directories. Under bash, the following command appends the path to the new BLAST bin directory to the existing PATH setting:

export PATH=$PATH:$HOME/ncbi-blast-2.6.0+/bin

The modified $PATH can be examined using echo

echo $PATH

Additionally, create a directory for blast databases e.g. blastdb

export BLASTDB=~/blastdb
