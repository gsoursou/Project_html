Blast Database

Download a DB from NCBI

We can download any BLAST databases available on the NCBI FTP site under the /blast/db directory with the command

ftp://ftp.ncbi.nlm.nih.gov/blast/db/dbname

where dbname is the name of the desired database to download.All the available downlowds are here. For example the command ftp://ftp.ncbi.nlm.nih.gov/blast/db/16SMicrobial.tar.gz will download the 16SMicrobial database.

Check downloaded database

In order to check your database you need set the BLASTDB variable as described here. To view the general information of your database use the command

blastdbcmd -info -db dbname

Examples

  • Print all records

    • blastdbcmd -db dbname -entry all -outfmt "%t %g %T"

  • Sort and display number of sequences for each species

    • blastdbcmd -db dbname -entry all -outfmt "%T"|sort|uniq -c|sort -n

    blastdbcmd -db dbname -entry all -outfmt "%T"|sort|uniq -c

  • Print only specific sequences

    • blastdbcmd -db dbname -entry 645321669 -outfmt "%t %g %T"

    blastdbcmd -db dbname -entry 112690,112672

You can refer to the BLAST database documentation for more details.
